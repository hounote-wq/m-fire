M-FIRE
Mortar Fire Integrated Resolution Engine
ร้อย ค.หนัก · กรมทหารม้าที่ 3
VER 10.4 · CLASSIFIED

ระบบประมวลผลการคำนวณวิถีกระสุนสำหรับเครื่องยิงลูกระเบิดแบบบูรณาการ พัฒนาขึ้นโดยกำลังพลกองร้อยเครื่องยิงหนัก กรมทหารม้าที่ 3 เป็น Web Application สำหรับศูนย์อำนวยการยิง (FDC) ช่วยคำนวณข้อมูลการยิงและจัดการเป้าหมายแบบอัตโนมัติ

กระสุนที่รองรับ

HE (M293)  Charge 0, 2, 4, 6, 8, 9  ·  ระยะสูงสุด 6,500 ม.
IL (M9602)  Charge 1, 2, 3, 4  ·  ระยะสูงสุด 6,900 ม.

5 แท็บหลักของระบบ
  • INPUT บันทึกพิกัด X/Y/ความสูง ของจุดตั้งปืน, ผตน. และเป้าหมาย รองรับ Preset ปืน 6, ผตน. 3, เป้าหมาย 6 ตำแหน่ง
  • FIRING แสดงผล Slant Range, Graphic Range, Azimuth, มุมสูง และแนะนำ Charge อัตโนมัติ
  • ADJUST รับคำสั่งปรับแก้จาก FO (เพิ่ม/ลด ระยะ, ซ้าย/ขวา, สูง/ต่ำ) แล้วคำนวณพิกัดเป้าหมายใหม่อัตโนมัติ
  • MAP แสดงกราฟิก 2D ตำแหน่งปืน, เป้าหมาย และผตน. บนตาราง MGRS Grid
  • LOG บันทึกประวัติการยิงแต่ละนัด (มุมทิศ, มุมสูง, Charge, ผล) และ Export PDF
วิธีใช้งาน
  • 1เข้าแท็บ INPUT → เลือก Preset ปืน → กรอกพิกัดจุดตั้งปืน
  • 2เลือก Preset เป้าหมาย → กรอกพิกัด → เลือกชนิดกระสุน HE / IL
  • 3กดปุ่ม คำนวณ ระบบจะสลับไปแท็บ FIRING DATA อัตโนมัติ
  • 4อ่านค่ามุมทิศ, มุมสูง, Charge นำไปออกคำสั่งให้ส่วนยิง
  • 5หาก FO แจ้งปรับแก้ ไปที่แท็บ ADJUST กรอกค่า ระบบจะอัปเดตเป้าอัตโนมัติ
  • ▲ สภาพอากาศ ระบบไม่ได้นำตัวแปรทางอุตุนิยมวิทยา (ลม, อุณหภูมิ, ความกดอากาศ) มาคำนวณร่วมด้วย อาจเกิดความคลาดเคลื่อนในการยิงระยะไกล
  • ▲ ภูมิประเทศ การคำนวณเป็นแบบ Point-to-Point เส้นตรง ไม่คำนึงถึงสิ่งกีดขวางหรือลักษณะชันของพื้นที่
  • ▲ ตารางยิง ข้อมูลถูก Hardcode เฉพาะกระสุน M293 และ M9602 เท่านั้น หากใช้กระสุนชนิดอื่น ผลที่คำนวณได้จะไม่ถูกต้อง
  • ▲ การเก็บรักษาข้อมูล เป็น Web App ข้อมูลที่กรอกไว้อาจสูญหายทันทีหากปิดหน้าเว็บหรือรีเฟรชเบราว์เซอร์
⚠ ผลการคำนวณทั้งหมดต้องได้รับการยืนยันจากนายทหาร FDC
